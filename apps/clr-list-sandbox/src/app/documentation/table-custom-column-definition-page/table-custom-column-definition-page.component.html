Table.CustomColumn

Example

Use this column definition, in case you need to render inside the cell some custom component. All input / output properties of the rendered component could be bound to the cell values, see example. It is suggested to use custom cells map to define custom cell types and associated with them rendered components across the whole application.

Property Type Description
id, title, sort, cls - The same as in Table.Column
customCell true Flag column as custom. You must provide for this column some html template.

First define colum

{{custom_column}}

And then in template

{{custom_column_template}}
