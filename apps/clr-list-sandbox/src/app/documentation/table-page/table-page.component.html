Property Type Description

Table aggregate row

Provides interface to define aggregation row on the table. The aggregated row will be shown first in the list and include aggregation data only from currently visible rows.

Example

For each column defined on the table you could provide aggregation function. This interface is the Map object `columnId : aggregationFunction for the column`. The aggregation function takes as a first param all values from all visible rows for this column and as second additional rows all visible rows.
