What is it ?

Set of angular libraries to facilitate development of data driven applications

When it could be usefull ?

  • For back office applications
  • For CRM like applications
  • In general when application contains a lot of lists, forms, input fields
  • Projects with dynamic forms

How it could help ?

  • Define forms declaratively via js objects
  • Define form complex layouts declaratively
  • Define complex behaviour of the input fields via Observable properties
  • Extend predefined set of inputs / layouts via configuration
  • Create wizards declaratively
  • Create lists including filter toolbox declaratively
  • Other utilitise such as modals and overlays

Other features

  • Default UX presentation layer implemented with clarity design system
  • Extensible and configurable
  • Its just angular libraries use where you need it alongside with you usual project implementation
